# Villages Management

## Overview

Manage informasi desa-desa di Kotagede.

## Fields

- **Key** (required): Unique identifier (e.g., "purbayan", "prenggan")
- **Title** (required): Nama desa
- **Image** (required): Path ke image
- **Blurb** (required): Deskripsi singkat desa

## Notes

- Hanya ada 5 villages: Purbayan, Prenggan, Basen, Kemasan, Jagalan
- Key harus unique dan lowercase
- Blurb digunakan untuk deskripsi singkat di explore page
